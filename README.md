Ytdlp-Apiは今度追加していく予定

## 技術スタック

- **バックエンド:** Python / FastAPI + httpx（非同期）+ Jinja2テンプレート
- **フロントエンド:** バニラJS・CSS・HTML（モジュールバンドラー不使用）
- **データソース:** Invidious API（`kuru-bana/yt-data` GitHub経由でインスタンスリスト管理）
- **画像プロキシ:** `wsrv.nl`（サムネイル・アイコンをWebP変換・キャッシュ）
- **ローカルストレージ:** 視聴履歴・チャンネル登録・お気に入り・プレイリストをブラウザに保存

## 主な機能

### 動画視聴
- **4種類の再生モード:**
  - **ストリームモード:** Invidiousから取得したストリームでの再生（デフォルト）
  - **HQモード:** 映像と音声を別ストリームで取得して同期再生（高画質）
  - **Nocookieモード:** `youtube-nocookie.com` iframe埋め込み
  - **Eduモード:** `youtubeeducation.com` 埋め込み（制限回避用）
- カスタムオーバーレイコントロール（画質選択・再生速度・ループ）
- シアターモード・キーボードショートカット対応

### ショート
- 縦スクロール型ショートプレーヤー（↑↓キー対応）
- コメント表示・メタ情報オーバーレイ
- 検索結果にショート専用棚を自動表示（バックグラウンド複数ページ取得）

### 検索・発見
- 検索候補サジェスト付きサーチ（履歴保存・クリア対応）
- フィルター: 並び順・投稿日・動画長・種別・機能・地域
- トレンド動画（カテゴリー別: 音楽・ゲーム・ニュース・映画）
- ハッシュタグ・ミックスプレイリスト対応

### チャンネル
- チャンネルプロフィール（バナー・アバター・登録者数）
- タブ切り替え: 動画・ショート・プレイリスト・コミュニティ・概要
- チャンネル内検索

### ライブラリ（ローカル保存）
- チャンネル登録管理
- 視聴履歴（最大1000件）
- お気に入り動画
- カスタムプレイリスト作成・編集
- インポート / エクスポート（JSON形式）

### ダウンローダー
- `/dl` ページでURLを解析して各フォーマットへの直リンクを提供
- 対応形式: 映像+音声・映像のみ・音声のみ・サムネイル・ストーリーボード

### その他
- **リンクリスト (`/links`):** 登録ミラーサイト一覧のオンライン状態・バージョンを確認（バージョン別・ドメイン別グループ表示）
- **チャット:** 外部チャット機能
- **起動時keepalive ping:** ホームアクセス時に外部サービスへ非同期ping送信（スリープ防止）

## ページ一覧

| パス | 説明 |
|---|---|
| `/` | トップ（トレンド動画） |
| `/watch?v=...` | 動画視聴ページ |
| `/shorts/{id}` | ショートプレーヤー |
| `/search?q=...` | 検索結果 |
| `/channel?id=...` | チャンネルページ |
| `/playlist?list=...` | プレイリスト |
| `/hashtag?tag=...` | ハッシュタグ検索 |
| `/mix?id=...` | ミックスプレイリスト |
| `/library` | ライブラリ |
| `/dl` | ダウンローダー |
| `/settings` | 設定 |
| `/links` | ミラーサイト一覧 |
| `/chat` | チャット |

## ファイル構成

```
main.py                        # FastAPIサーバー・プロキシロジック
requirements.txt               # Pythonパッケージ依存
templates/
  base.html                    # 共通レイアウト
  index.html / watch.html / search.html ... # 各ページ
  static/
    css/                       # CSS分割ファイル（base/global/search/channel/watch/library/settings/shorts）
    js/                        # JS分割ファイル（utils/components/library/header/page-*）
```

## 起動方法

```bash
pip install -r requirements.txt
uvicorn main:app --host 0.0.0.0 --port 5000
```

## ユーザー設定

- キャッシュTTL: 検索結果5分（sessionStorage）
- デフォルト再生速度・音量・自動次再生・ループはブラウザのlocalStorageに保存
